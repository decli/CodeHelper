# 取件码助手（CodeHelper）

一款为老人场景设计的本地短信取件码助手。

## 功能

- 首屏直接读取短信并提取取件码，默认展示最近 12 小时
- 支持 12 小时、1 天、2 天、5 天快速筛选
- 支持双击卡片跳转到对应短信
- 左滑红色抽屉删除，强制刷新后会以灰色卡片和“已删除”标签重新载入
- 支持自定义取件码提取规则，默认内置 `取件码` 与 `凭` 两类规则
- 全程离线运行，不申请网络权限

## 构建

```bash
./gradlew assembleRelease
```

推送到 `main` 后，GitHub Actions 会执行构建校验并上传 APK artifact。
推送 `v*` 标签后，GitHub Actions 会在 Releases 页面创建正式版本并附带可下载 APK。
